# Snapshot Topic display data on Meeting completion

When a Meeting becomes completed, each agenda appearance preserves the Topic values needed to render that Meeting's historical view. Every appearance snapshots the Topic name and responsible-user display name, while New membership appearances additionally snapshot their process status, signal, and godparents. Planned and in-progress Meetings continue using live editable Topic data, while completed Meetings render snapshots so later Topic changes cannot rewrite the recorded Meeting.
